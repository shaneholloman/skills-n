# Skill Categories

n-skills organizes skills by function into these categories:

## tools

CLI tools and utilities that extend agent capabilities.

**Examples:**
- zai-cli - Z.AI vision, search, reader, and GitHub exploration
- API clients
- File processors

## development

Language-specific development assistance.

**Examples:**
- TypeScript expert guidance
- Python debugging helpers
- Framework-specific patterns

## productivity

Workflow automation and efficiency tools.

**Examples:**
- Git workflow automation
- Task management
- Code review assistance

## automation

Browser, CI/CD, and system automation.

**Examples:**
- Browser automation with Playwright
- CI/CD pipeline helpers
- System administration tasks

## data

Databases, data processing, and analysis.

**Examples:**
- SQL query assistance
- Data transformation
- Analytics helpers

## documentation

Documentation, diagrams, and specifications.

**Examples:**
- API documentation generation
- Diagram creation
- Specification writing

## Adding a New Category

If your skill doesn't fit existing categories, open an issue to propose a new one. Include:

1. Category name (lowercase, singular)
2. Description
3. 2-3 example skills that would belong
